    Каждое приложение Android выполняется в собственном процессе, со своим собственным экземпляром виртуальной машины Dalvik (или ART). ВМ Dalvik - DVM

    DVM - java c изменениями, т.к. стандартная машина java использует стековый подход, при вызове функции в стеке выделяется память и т.д. и т.п., повышенное потребление памяти - повышенное потребление электропитания

    DVM - память не стековая, память - регистровая Dalvik - деревня в Исландии, где жила бабушка одного из разработчиков Android

    DVM - компиляция при запуске программы

    ART VM - компиляция при установке .dex - dalvik execute
